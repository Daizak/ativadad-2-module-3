Propiedad Valor Inicial Descripción
color black Define el color del texto.
font-family serif Especifica la fuente del texto.
font-size medium Establece el tamaño de la fuente.
font-weight normal Controla el grosor del texto (normal, bold, etc.).
font-style normal Define el estilo de la fuente (normal, italic, oblique).
font-variant normal Controla la variante de la fuente (normal o small-caps).
font inherit Es una propiedad abreviada que establece varias propiedades de fuente.
text-align left Alinea el texto (left, right, center, justify).
line-height normal Establece la altura de línea, afectando el espacio entre líneas de texto.
text-decoration none Define la decoración del texto (underline, overline, line-through, none).
text-transform none Controla la transformación del texto (uppercase, lowercase, capitalize).
vertical-align baseline Alinea el texto verticalmente en relación con su contenedor.
text-indent 0 Establece la sangría del primer renglón de un bloque de texto.
letter-spacing normal Controla el espacio entre letras.
word-spacing normal Controla el espacio entre palabras.
white-space normal Controla cómo se manejan los espacios en blanco dentro del texto.
Característica Propiedades Descripción
Tamaño, Color y Decoración color, font-size, text-decoration Controla el color, tamaño y decoración de los enlaces.
Decoración Personalizada background-color, border, padding Permite personalizar la apariencia de los enlaces con fondo, bordes y relleno.
Característica Propiedades Descripción
Anchura y Altura width, height Establece las dimensiones de la imagen.
Eliminar el Borde border: none Elimina el borde alrededor de la imagen.
Propiedad Valor Inicial Descripción
list-style-type disc Define el tipo de marcador de la lista (disc, circle, square, none).
list-style-position outside Controla la posición de los marcadores (inside o outside).
list-style-image none Permite usar una imagen como marcador en lugar del estilo predeterminado.
list-style none Propiedad abreviada que establece las propiedades de estilo de lista.
Propiedad Se Aplica a Valor Inicial Descripción
border-collapse table separate Controla si los bordes de las celdas se colapsan o se separan.
border-spacing table 0 Establece el espacio entre los bordes de las celdas.
empty-cells table show Controla la visibilidad de las celdas vacías.
caption-side table top Define la posición del título de la tabla (top, bottom, left, right).
